Setelah sekian lama hanya menjadi wacana untuk mulai menulis, akhirnya kemarin ketika ada promo domain .id saya paksain untuk membeli domain ini. Awalnya memang sudah lama punya keinginan untuk membuat blog berdua, cuma karena belum sempat ya jadinya cuma sebatas angan. Kalau udah dibeli domain nya gini kan mau ga mau jadi harus dimanfaatkan, kan sayang udah beli domainnya.

Cuma setelah beli domain masih ada satu lagi masalahnya 😅, websitenya mau di host dimana?, awalnya mau pakai hosting Indonesia yang murah-murah gitu, tapi kok kayaknya sayang ya, orang isinya cuma tulisan-tulisan ga jelas juga nantinya, masa saya mesti bayar buat begituan :v, sempat terpikir juga buat pakai cloud kayak [Heroku](http://www.heroku.com) gitu juga, cuma kok kayaknya ribet juga :v padahal gratis ya walaupun ada batasan dyno hours nya, terus kepikiran buat pakai [openshift](https://openshift.redhat.com) juga, udah di pake eh ternyata gratisnya cuma beberapa hari doang :v, ya udah ganti lagi pakai [DigitalOcean](http://www.digitalocean.com) aja kali ya, kan lumayan dapet server juga kalau nanti mau lakuin hal-hal lain yang ga mungkin di laptop sendiri, kayak buat server [MQTT](https://en.wikipedia.org/wiki/MQTT) atau [XMPP](https://en.wikipedia.org/wiki/XMPP), atau dibikin buat ngitung model [TensorFlow](https://www.tensorflow.org) 😅, kan kasihan laptopnya kalau dipake buat komputasi tensorflow berhari-hari (gara-gara ga punya GPU yg sangar 😬).

Setelah dipikir-pikir ulang lagi, akhirnya kepikiran kenapa ga pake [static webnya github](https://pages.github.com) aja kan gratis 😂, bisa di custom domain juga. Dan akhirnya jadilah blog ini dibuat pake static webnya github. Ya udah itu sedikit cerita gimana ini blog muuncul ke permukaan dunia maya ini, ditengah kebimbangan juga kenapa ga pake [Medium](https://medium.com) aja :v, kan lagi ngetren tu orang-orang nulis di medium. Tapi ya sudahlah, kita pakai blog sendiri aja, dan selamat menikmati tulisan-tulisan luar biasa dari kami Zelory di kemudian hari 😁.
